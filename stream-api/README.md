# Stream API practice

Модуль содержит 10 задач уровня middle для подготовки к Java-интервью.

Запуск тестов:

```bash
./gradlew :stream-api:test
```

Сейчас тесты должны быть красными: методы в `StreamTasks` оставлены как заглушки. Твоя цель - реализовать методы через Stream API так, чтобы тесты стали зелеными.

## Задачи

1. `normalizeAndDeduplicateWords` - нормализовать слова: убрать пробелы по краям, привести к lowercase, удалить пустые строки и дубликаты, сохранив порядок первого появления.
2. `topNFrequentWords` - найти топ N самых частых слов. Сортировка: частота по убыванию, затем слово по алфавиту.
3. `activeEmployeeNamesByDepartment` - сгруппировать активных сотрудников по отделу. Внутри отдела имена должны быть отсортированы по зарплате по убыванию, затем по имени.
4. `averageSalaryByDepartment` - посчитать среднюю зарплату активных сотрудников по каждому отделу.
5. `secondHighestPaidEmployee` - найти сотрудника со второй уникальной по величине зарплатой.
6. `totalAmountByCustomer` - посчитать общую сумму заказов по каждому клиенту с учетом цены и количества товаров.
7. `mostExpensiveProductByCategory` - найти самый дорогой товар в каждой категории. При равной цене выбрать товар с лексикографически меньшим названием.
8. `lastTransactionByAccount` - для каждого аккаунта найти последнюю транзакцию по времени. При одинаковом времени выбрать транзакцию с большим `id`.
9. `countLogLevels` - распарсить строки логов и посчитать уровни `INFO`, `WARN`, `ERROR`, `DEBUG`. Некорректные строки игнорировать.
10. `topTags` - собрать теги из статей, нормализовать их, удалить пустые и вернуть топ N по частоте. Сортировка: частота по убыванию, затем тег по алфавиту.
