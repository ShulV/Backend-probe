#### ТЗ
Нужно реализовать "кэш", который хранит лонги по индексу.
Размер кэша — 10 элементов.


Реализует два метода:
void bulkUpdate(Updater updater)
и
long[] bulkRead(int[] indices)

* bulkUpdate обновляет значения пачкой в текущем состоянии кэша, in-place.
* bulkRead получает пачкой необходимые лонги из кэша по индексам.

#### Условия:

1. Есть N (константа) потоков, которые кэш читают.
2. Есть 1 поток, который кэш обновляет.
3. Читатель должен быть защищён от dirty-read.
   То есть, если происходит мутация A -> B,
   то читатель должен видеть только конечное состояние
   (A или B), но никогда промежуточное.
4. Чтение должно быть неблокирующим.
5. Запись может быть блокирующей.
6. Входные данные можно считать всегда валидными
   (не null; индексы только от 0 до 9 включительно).
